Front-End Projects

Here are a few note-worthy front-end development projects I've completed while studying various libraries:

D3 Tree graph

A project on graphing tree graphs using the .D3 library. Includes the option to toggle between three data sets, as well as a colour shuffler to mix up the colours used for the legend.

Pomodoro Timer

A simple timer for variable work-break lengths, initialised to 25 minutes and 5 minutes respectively. Plays a funky bongo rhythm once the timer is done and updates the text and icon shown depending on state.

Calculator

An extension of a simpler project prompt, this calculator also has a few additional methods such as finding the square root or converting to percentages, as well as a few ease-of-use methods such as toggling whether an input number is negative or undoing the latest input.
